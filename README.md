# 🧠 AI Habit Tracker + Advisor

A simple AI-powered habit tracking app that analyzes your consistency and provides smart suggestions.

## 🚀 Features
- Track daily habits
- Calculate streaks
- Completion rate analysis
- AI-based advice

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas

## ▶️ Run Project
pip install -r requirements.txt  
streamlit run app.py
